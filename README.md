# Credit Risk Modelling using Machine Learning

## Overview

This project focuses on predicting loan approval and analyzing credit risk using Machine Learning techniques. The model analyzes applicant information such as income, education, employment status, credit history, and loan amount to determine loan approval possibilities.

The project demonstrates:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Model Evaluation
- Data Visualization
